# STEM Visualizer

Interactive visual demos for:
- Algorithms: Bubble sort step-by-step with speed controls.
- Math: Live function grapher (sine, parabola, gaussian, damped cosine).
- Physics: Projectile motion simulator with adjustable angle, speed, and gravity.

## Run

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
